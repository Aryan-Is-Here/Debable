# DebateMatch Blueprint

This repository contains the planning documents for building the DebateMatch MVP using AI-assisted development.

Suggested implementation order:

1. Product Vision
2. PRD
3. Architecture
4. Database
5. API
6. UI/UX
7. Roadmap
8. Git Workflow
9. AI Design
10. Vibe Coding Playbook

Never ask an AI to build the whole project. Build one module at a time.
